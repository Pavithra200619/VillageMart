# 🛒 VillageMart

<p align="center">
  <b>A Multi-Vendor Grocery Marketplace</b><br>
  Inspired by Blinkit, BigBasket & JioMart
</p>

---

## 📌 Overview

VillageMart is a full-stack multi-vendor grocery marketplace where customers can purchase products from multiple local sellers, sellers can manage their products and orders, and an admin can manage the entire platform.

The project aims to provide a seamless online shopping experience while helping local vendors sell their products digitally.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/Pavithra200619/VillageMart.git
```

---

## Backend Setup

```bash
cd backend

npm install

npm start
```

---

## Frontend Setup

```bash
cd frontend

npm install

npm run dev
```
